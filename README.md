## Catatan Pembelajaran
- Abstraksi membuat fokus pada apa yang dilakukan sebuah kelas tanpa harus paham bagaimana caranya.
- Polymorphism memungkinkan satu metode dipakai untuk banyak tipe objek berbeda asalkan mereka punya interface yang sama.
- Gabungan abstraksi dan polymorphism membuat kode gampang diubah atau ditambah fitur tanpa merusak bagian lain.
- Membantu membuat desain program lebih rapi dan memudahkan kerja tim karena aturan (interface) jelas.
